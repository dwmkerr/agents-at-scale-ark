# Dashboard

The Ark dashboard provides a visual interface for managing your agents, teams, and models.

## Access the Dashboard

Run the `ark dashboard` command:

```bash
ark dashboard
```

The dashboard will open in your browser:

![Ark Dashboard](/ark-dashboard-screenshot.png)

The dashboard can be used to create models, agents, teams, tools, run queries, check events and more.
